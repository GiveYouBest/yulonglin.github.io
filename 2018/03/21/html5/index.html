Html5

1.行内标签

行内标签特征：(1) 设置宽高无效

​ (2) 对margin仅设置左右方向有效，上下无效; padding设置上下左右都有效，即会撑大空间

​ (3) 不会自动换行

常见的行内标签

span
a
em
strong
b
i
u
label
br
code
cite

2. 块标签

块标签特征: (1) 能够设置宽高

​ (2) margin和padding的上下左右无效

​ (3) 可以自动换行

​ (4) 多个块状标签写到一起，默认排列方式从上到下

​ (5) 独自占一行

常见的块标签

div
p
h1-h6
ol(有序列表)
ul(无序列表)
dl
table
address
form
legend
li

3. 行内块标签

行内块状标签特征: (1) 不自动换行

​ (2) 能够识别宽高

​ (3) 默认排列方式从左到右

​ (4) 一行多个

常见的行内块

img
input
td

常见自闭和标签

input
meta
link
br
hr
img

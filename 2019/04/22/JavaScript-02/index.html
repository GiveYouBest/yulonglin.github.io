1.运算符

1.1运算符的分类

运算符(operator)是用于实现赋值、比较和执行算数 运算等功能的符号。

JavaScript中常用的运算符有：

算数运算符 ：主要有 + - * / % 加 减 乘 除 取余

递增和递减运算符 ++num,num++

比较运算符 > < !等

逻辑运算符 与或非 && || !

赋值运算符 = += -=等

2.分支流程控制

2.1分支结构

由上到下执行代码的过程，根据不同的条件，执行不同的路径代码，从而得到不同的结果。

JS语言提供了两种分支结构语句：if语句、switch语句

  • if 语句
  • if else语句（双分支语句）
  • if else if 语句(多分支语句)

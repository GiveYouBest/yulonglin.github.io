Arrary.isArrary()用于判断一个对象是否为数组，isArray()是HTML5中提供的方法

添加删除数组元素方法

方法名 说明 返回值
push() 末尾添加一个或多个元素，注意修改原数组 返回新的长度
pop() 删除数组最后一个元素，长度减1 返回擅长的元素的值
unshift() 向数组的开头添加一个或更多元素 返回新的长度
shift() 删除数组的第一个元素，数组长度减1 返回第一个元素的值

注意：push、unshift为增加元素方法；pop、shift为删除元素的方法

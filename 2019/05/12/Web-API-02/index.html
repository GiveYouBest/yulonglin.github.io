1.排他操作

1.1 排他思想

思路：

如果有同一组元素，我们想要某一个元素实现某种样式，需要到循环排他思想：

​ 1.所有元素全部清除样式(干掉其他人)

​ 2.给当前元素设置样式(留下我自己)

​ 3.注意顺序不能颠倒，首先干掉其他人，再设置自己。

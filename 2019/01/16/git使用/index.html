git

初始化git仓库

这个仓库会存放，git对我们项目代码进行备份的文件

命令 git init —-初始化

对git中设置当前使用的用户 —-

git config –global user.name “用户名” –配置用户名

git config –global user.email “邮箱” –配置邮箱

1.把代码存储到 .git仓库中

第一步:git add 文件所在位置

第二步: git commit -m “” ——- -m:message 描述完成了什么功能

查看状态 git status

多个文件 使用git add ./ ———— 它会将修改过的文件放到大门口

一次性把文件放入 仓库中 git commit –all -m “”

2.设置忽略文件

.gitignore，在这个文件中可以设置要忽略的文件或目录 被忽略的文件不会被提交到仓库中

建立一个.gitignore文件在文件中写上要忽略文件的路径以/开头 一行写一个路径

*·写法 /.idea 会忽略.idea文件 /js 会忽略js目录里的所有文件
